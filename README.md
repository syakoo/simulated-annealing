# Usage

First, create a venv and activate it.

```console
$ python -m venv venv
$ source venv/bin/activate
```

Then, you should import packages from `requirements.txt`

```console
$ python -m pip install -r requirements.txt
```

Finally, run the following command.

```console
$ python main.py
```
